# IP-cores-para-algoritmos-de-ordenamiennto-en-FPGA #

El presente trabajo se propone el diseño, implementación y validación de un conjunto de IP-Cores de algoritmos de ordenamiento destinados a su ejecución sobre dispositivos FPGA.  

El objetivo principal consiste en desarrollar módulos reutilizables y parametrizables que permitan adaptar distintas características de arquitectura según los requerimientos de la aplicación, tales como minimización del área ocupada, reducción de la cantidad de ciclos de reloj, cantidad de datos a procesar, ancho de palabra, y nivel de paralelismo implementado. 

El proyecto contempla el desarrollo de arquitecturas descritas en HDL, su simulación funcional, síntesis y caracterización sobre FPGA, así como la validación de su funcionamiento mediante aplicaciones de prueba.
